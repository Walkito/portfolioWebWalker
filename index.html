Mais Vida

Este se trata do meu primeiro projeto pessoal em que utilizei REST API com Springboot e JPA. Já havia feito alguns exercícios em curso sobre REST API, contudo, este foi o meu primeiro projeto do zero ao qual fiz sozinho.

Esta REST API simula um software para entidades da saúde, onde é possível cadastrar pacientes, especialidades e agendar consultas. Tudo isso, salvando, editando e excluindo de um banco de dados local.

Este projeto possuí apenas o Back-End com integração com o Banco de Dados. Existe um outro projeto ao qual já integrei Front-End e Back-End.

Tecnologias Utilizadas: Java, JPA, Maven, MVC, MySQL, REST API, Springboot

Gestor de Contatos

Este projeto se trata de um teste prático que realizei para aplicar à uma oportunidade em uma empresa. O objetivo era construir um sistema simples, mas totalmente completo: Uma REST API onde fosse possível cadastrar pessoas e contatos destas pessoas. Contudo, cada contato só poderia estar relacionado com apenas uma pessoa, enquanto as pessoas poderiam ter vários contatos.

Tudo isto deveria ser feito utilizando Java como Back-End, Angular como Front-End e o PostgreSQL como Banco de Dados para persistir os dados. Este projeto foi a minha primeira integração entre Back-End e Front-End, além de ser minha primeira experiência com Angular também.

Tecnologias Utilizadas: Angular, Java, JPA, Maven, MVC, PostgreSQL, REST API, Springboot
