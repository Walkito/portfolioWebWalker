Termo

Este projeto é o meu TCC da faculdade de Ciência da Computação. A proposta era sobre criar um jogo mobile para Android e posteriormente, disponibi- lizá-lo na Google Play. Tendo isto em mente decidimos recriar o famoso jogo da Web "Termo", só que para Mobile.

O Termo nada mais é de que um jogo da forca, contudo, você possuí apenas 6 tentativas para acertar uma palavra de 5 letras. Em nosso projeto, também adicionamos uma pequena historinha para complementar o jogo. O mesmo está em sua primeira versão.

O planejamento é até o final do ano finalizar o jogo e publicá-lo na Google Play.

Tecnologias Utilizadas: Android Studio, Java

Mais Vida

Este projeto se trata do meu primeiro projeto pessoal em que utilizei REST API com Springboot e JPA. Já havia feito alguns exercícios em curso sobre REST API, contudo, este foi o meu primeiro projeto do zero ao qual fiz sozinho.

Se trata de uma REST API simulando um software para entidades da saúde, onde é possível cadastrar pacientes, especialidades e agendar consultas. Tudo isso, salvando, editando e excluindo de um banco de dados local.

Por enquanto, só possuí o back-end, mas já estou estudando formas de implementar um front-end para realizar esta integração. Em breve o projeto será finalizado.

Tecnologias Utilizadas: Java, JPA, Maven, MVC, MySQL, REST API, Springboot
